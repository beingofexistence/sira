## 位置

合法的位置样式类名: `top` `bottom` `left` `right` `top-left` `top-right` `bottom-left` `bottom-right` `left-top` `right-top` `left-bottom` `right-bottom`

使用指定的值在此行`bottom`类的位置修改下拉列表位置。

```html {3} /bottom/
<div class="dropdown success">
  <label class="btn solid" tabindex="0">Open Dropdown</label>
  <div class="menu bottom">
    <a class="item text-sm">Profile</a>
    <a class="item text-sm" tabindex="-1">Account settings</a>
    <a class="item text-sm" tabindex="-1">Subscriptions</a>
  </div>
</div>
```
